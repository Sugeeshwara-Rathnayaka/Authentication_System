A **full-stack MERN** authentication system featuring **JWT-based authentication**, **email OTP verification**, **password reset**, and secure user management. Frontend and backend can be deployed separately (Render, Netlify, Vercel).

---

## üî• Features

- User **Sign Up** with email & password
- **Email OTP verification** for account activation
- **Login** with JWT
- **Password reset** via email OTP
- **Protected routes** for authenticated users
- Responsive UI built with **Tailwind CSS**
- Toast notifications for success/error messages
- Cross-origin deployment ready
- JWT handling for deployed environments (avoids 401 errors)
